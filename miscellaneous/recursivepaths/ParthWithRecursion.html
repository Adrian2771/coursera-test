This application calculates and displays the possible paths from the top-left corner to the bottom-right corner of a m * n matrix.

A path is deemed as 'possible' if it can be covered by moving only vertically (top down) and horizontally (left to right). Since for a 17 x 17 matrix, there are 601080390 possible paths, caution should be taken when using higher dimensions. Printing the possible paths for a matrix with more than 10 columns x 10 rows is not recommended, since the number of possible paths in this case is 48620 and the number of paths for the next upper case (11 x 11) is 184756.
